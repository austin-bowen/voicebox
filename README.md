# voicebox
Python text-to-speech library with built-in voice effects

## Setup

1. Setup virtual environment, e.g.
   1. `python3 -m venv venv`
   2. `. venv/bin/activate`
2. Install dependencies:
   1. `pip install -U pip`
   2. `pip install -r requirements.txt`
   3. For development:\
      `pip install -r requirements.test.txt`
